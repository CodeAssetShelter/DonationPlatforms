# DonationPlatforms
 숲을 제외한 치지직, 투네이션과 연결하는 유틸
<br>
# 주의사항
 이 프로젝트는 WebSocket4Net 및 RestAPI, Newtonsoft.Json 을 사용합니다.<br>
 프로젝트의 Nuget 관리자에서 반드시 받아주세요.<br>
 https://www.nuget.org/packages/WebSocket4Net/<br>
 버전은 상관없습니다.<br>
